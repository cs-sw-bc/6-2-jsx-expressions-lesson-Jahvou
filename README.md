# JSX Lesson — CircuitStream Front End Development (Lesson 2)

## What You'll Learn

Three progressive guides covering core JSX patterns:

1. [JSX Expressions](docs/01-jsx-expressions.md) — Using `{}` to embed JavaScript in JSX
2. [Conditional Rendering](docs/02-conditional-rendering.md) — if/else, ternary, `&&`, early returns
3. [List Rendering](docs/03-list-rendering.md) — `.map()`, `key` prop, filter/sort chaining

## Quick Start

```bash
npm install
npm run dev
```

## Project Structure

- `src/components/JSX_Expressions_Interactive_Lesson.jsx` — Interactive lesson (currently rendered)
- `src/components/JSXExpressionChallenges.jsx` — Coding challenges component (ready to wire in)

## Commands

```bash
npm run dev       # Start Vite dev server (hot reload)
npm run build     # Production build
npm run preview   # Preview production build locally
npm run lint      # Run ESLint
```
